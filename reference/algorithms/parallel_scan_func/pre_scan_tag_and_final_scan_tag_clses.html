pre_scan_tag クラスと final_scan_tag クラス

概要

parallel_scan のフェーズを識別する型。

ヘッダー

 #include "tbb/parallel_scan.h"

構文

struct pre_scan_tag; 
struct final_scan_tag;

説明

pre_scan_tag 型と final_scan_tag 型は、parallel_scan とともに使用される仮の型です。これらの型が operator() の署名でどのように使用されるかについては、「parallel_scan テンプレート関数」セクションの例を参照してください。

メンバー

namespace tbb {

    struct pre_scan_tag {
        static bool is_final_scan();
    };

    struct final_scan_tag {
        static bool is_final_scan();
    };

}
次の表は、このテンプレート・クラスのメンバーの詳細な情報を提供します。
メンバー 説明
bool is_final_scan()

final_scan_tag の場合は true、その他の場合は false。
