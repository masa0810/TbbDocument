flow_control クラス

概要

入力の最後に達したことを示すため、複合フィルターの最初のフィルターを有効にします。

ヘッダー

#include "tbb/pipeline.h"

構文

class flow_control;

説明

parallel_pipeline テンプレート関数は、flow_control オブジェクト fc を filter_t の入力ファンクターに渡します。入力ファンクターは入力の最後に達すると、fc.stop() を呼び出し、ダミー値を返します。

メンバー

namespace tbb {
    class flow_control {
    public:
        void stop();
    };
}
