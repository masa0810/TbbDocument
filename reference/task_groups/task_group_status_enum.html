task_group_status 列挙値

task_group_status は、task_group のステータスを表します。

メンバー

namespace tbb {
                    enum task_group_status {
                        not_complete, // キャンセルされずグループのすべてのタスクが完了しなかった 
                        complete,     // キャンセルされずグループのすべてのタスクが完了した
                        canceled      // タスクグループがキャンセルのリクエストを受け取った
     };
 }
