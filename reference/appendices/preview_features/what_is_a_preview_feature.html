プレビュー機能について

プレビュー機能は、インテル® TBB のコンポーネントで、現在、開発者の皆様に初期フィードバックへのご協力をお願いしています。プレビュー機能に関するご意見、ご要望、質問を www.threadingbuildingblocks.org までお寄せください。

プレビュー機能の主な特性:

  • 明示的に有効にする必要があります。デフォルトではオフです。

  • 高品質な実装を目的としています。

  • 将来の製品の継続性および互換性は保証されていません。

  • 正当性アナライザー、プロファイラー、デバッガーのようなツールでは、サポートが全く提供されていないか、限定されています。

注意

プレビュー機能は、将来変更されることがあります。ライブラリーの将来のリリースで削除または根本的に変更される可能性があります。プレビュー機能への変更は、通常の廃止および削除プロセスが必要ありません。製品コードではプレビュー機能を使用しないでください。
