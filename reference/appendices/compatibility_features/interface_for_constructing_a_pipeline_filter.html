パイプライン・フィルター構築用のインターフェイス

インテル® TBB のいくつかのリリースを経て、フィルターを構築するインターフェイスが進化しました。

次の表は、インターフェイスの変更点を示します。
メンバー 説明
filter::filter( bool is_serial )

is_serial が true の場合、シリアル・インオーダー・フィルターを構築します。is_serial が false の場合、並列フィルターを構築します。これらのコンストラクターの代わりに filter( filter::mode) コンストラクターが使用されます。
filter::serial

フィルターモード filter::serial は filter::serial_in_order に名前が変わりました。新しい名前は filter::serial_out_of_order モードとより区別がつきやすくなりました。
