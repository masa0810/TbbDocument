Allocator コンセプト

インテル® TBB で実装されるアロケーターの Allocator コンセプトは ISO C++ 標準の表 32 の「Allocator 要件」に似ていますが、ISO C++ コンテナーの使用にはさらなる保証が必要です。下記の表は、Allocator コンセプトを要約したものです。ここで、A と B はアロケーター・クラスのインスタンスを表します。
