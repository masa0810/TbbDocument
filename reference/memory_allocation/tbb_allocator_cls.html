tbb_allocator テンプレート・クラス

概要

利用可能な場合はスケーラブルなメモリー割り当てテンプレート・クラス。利用できない場合は非スケーラブル。

構文

template<typename T> class tbb_allocator

ヘッダー

#include "tbb/tbb_allocator.h"

説明

tbb_allocator は、インテル® TBB の malloc ライブラリーが利用可能な場合はこれを使用してメモリーの割り当てと解放を行います。利用できない場合は、通常の malloc と free を使用します。

ヒント

インテル® TBB の malloc ライブラリーが使用されているかどうかを確認するには、TBB_VERSION 環境変数を 1 に設定します。
