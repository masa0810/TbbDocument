イテレーター

enumerable_thread_specific テンプレート・クラスは、コンテナーのすべての要素のセットで反復可能な、ランダムアクセス・イテレーターをサポートします。

次の表は、このテンプレート・クラスのメンバーの詳細な情報を提供します。
メンバー 説明
iterator begin()

戻り値: 要素のセットの最初を指す iterator。
iterator end()

戻り値: 要素のセットの最後を指す iterator。
const_iterator begin() const

戻り値: 要素のセットの最初を指す const_iterator。
const_iterator end() const

戻り値: 要素のセットの最後を指す const_iterator。
