empty_task クラス

概要

何もしないことを表すタスクのサブクラス。

構文

class empty_task;

ヘッダー

#include "tbb/task.h"

説明

empty_task は、何も行わないタスクです。継続がプレデセッサーの完了を待つ以外に何もすべきでない場合、親タスクの継続として利用できます。

メンバー

 namespace tbb {
        class empty_task: public task {
            /* オーバーライド */ task* execute() {return NULL;}
        };  
 }
