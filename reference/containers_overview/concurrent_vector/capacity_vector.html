キャパシティー

次の表は、このテンプレート・クラスのメンバーの詳細な情報を提供します。
メンバー 説明
size_type size() const

戻り値: ベクトルの要素の数。結果には、拡張メソッドの同時呼び出しによって割り当てられているものの、まだ構築中の要素も含まれます。
bool empty() const

戻り値: size()==0。
size_type capacity() const

戻り値: ベクトルがより多くのメモリーを割り当てることなく拡張できる最大サイズ。

注

std::vector と異なり、concurrent_vector は、より多くのメモリーを割り当てられる場合に既存の要素を移動しません。
size_type max_size() const

戻り値: ベクトルの可能な最大サイズ。
