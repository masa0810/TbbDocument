アクセス

注意

このセクションで説明されているメソッドは、同時拡張用のメソッドとして、同じベクトルで同時に呼び出すことができる場合があります。ただし、同時に構築された要素への参照が返されることがあります。

次の表は、このテンプレート・クラスのメンバーの詳細な情報を提供します。
メンバー 説明
reference operator[]( size_type index )

戻り値: 指定したインデックス要素の参照。
const_refrence operator[]( size_type index ) const

戻り値: 指定したインデックス要素の const 参照。
reference at( size_type index )

戻り値: 指定したインデックスの要素への参照。

スロー : index >= size() の場合は std::out_of_range。
const_reference at( size_type index ) const

戻り値: 指定したインデックスの要素への const 参照。

スロー: index >= size() またはベクトルの無効な部分に対する index の場合は std::out_of_range。
reference front()

戻り値: (*this)[0]
const_reference front() const

戻り値: (*this)[0]
reference back()

戻り値: (*this)[size()-1]
const_reference back() const

戻り値: (*this)[size()-1]
