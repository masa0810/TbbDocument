サイズおよびキャパシティー

次の表は、このテンプレート・クラスのメンバーの詳細な情報を提供します。
メンバー 説明
bool empty() const

戻り値: size()!=0。
size_type size() const

戻り値: *this のアイテムの数。

注意

現在の実装は、時間が O(1) かかりますが、将来の実装では O(P) になる可能性があります。ここで、P はハードウェア・スレッドの数です。
size_type max_size() const

戻り値: *this が保持できるアイテムの数の上限。

注意

コンテナーが実際に保持できる数よりも大きいことがあります。
