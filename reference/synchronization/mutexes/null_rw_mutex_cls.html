null_rw_mutex クラス

概要

ReaderWriterMutex コンセプトをモデル化するが何も行わないクラス。

構文

class null_rw_mutex;

ヘッダー

#include "tbb/null_rw_mutex.h"

説明

null_rw_mutex は、ReaderWriterMutex コンセプトをモデル化しますが、何も行いません。ReaderWriterMutex を想定しているが、そのインスタンスで排他制御が実際に必要ないテンプレートを扱う場合に便利です。
