critical_section

概要

PPL 互換の mutex。

構文

class critical_section;

ヘッダー

#include "tbb/critical_section.h"

説明

critical_section は、PPL の critical_section を実装します。このクラスの機能は、tbb::mutex の機能のサブセットです。

メンバー

namespace tbb {
        class critical_section {
        public:
            critical_section();
            ~critical_section();
            void lock();
            bool try_lock();
            void unlock();
     
            class scoped_lock {
            public:
                scoped_lock( critical_section& mutex );
                ~scoped_lock();
            };
       };
    }
