graph_node クラス

概要

すべてのグラフノードの基本クラス。

構文

class graph_node;

ヘッダー

#include "tbb/flow_graph.h"

説明

graph_node クラスは、すべてのフローグラフのノードの基本クラスです。仮想デストラクターを使用して、フローグラフのノードを graph_node のポインターで破棄します。例えば、vector< graph_node * > は、後で破棄する必要があるフローグラフのノードのアドレスを保持するために使用します。

メンバー

namespace tbb {
namespace flow {
 
class graph_node {
public:
    explicit graph_node( graph &g );
    virtual ~graph_node() {}
};
 
}
}
