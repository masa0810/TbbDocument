Microsoft* Visual Studio* プロジェクト用統合プラグイン

プラグインにより、インテル® TBB を Microsoft* Visual Studio* プロジェクトへ簡単に統合できます。プラグインは、http://threadingbuildingblocks.org (英語) の [Download] > [Plug-Ins] からダウンロードできます。プラグインを使用すると、Microsoft* Visual C++* プロジェクトに次の項目が追加されます。

  • インテル® TBB ヘッダーファイルのパス

  • インテル® TBB ライブラリーのパス

  • リンクする特定のインテル® TBB ライブラリー

  • インテル® TBB 固有の設定

プラグインは、Microsoft* Visual Studio* 2005、2008、2010 で作成された C++ プロジェクトに利用できます。

この機能を使用するには、ダウンロードしたパッケージ msvs_plugin.zip を展開し、README.txt の説明に従ってインストールしてください。
