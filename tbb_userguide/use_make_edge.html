make_edge と remove_edge の使用

エッジの作成と削除には基本的なガイドラインがあります。

  • make_edge と remove_edge を使用する。

  • register_successor と register_predecessor を使用しない。

  • remove_successor と remove_predecessor を使用しない。

トポロジーと通信する場合は、flow::make_edge 関数と flow::remove_edge 関数のみを使用してください。ランタイム・ライブラリーは、これらのエッジを作成するために sender<T>::register_successor のようなノード関数を使用しますが、これらの関数は直接呼び出すべきではありません。ランタイム・ライブラリーは、ランタイムにトポロジー上の最適化を実装するために、これらのノード関数を直接呼び出します。
